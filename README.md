# 🛠️ JSON 工具箱

完全免费的在线 JSON 格式化、验证和转换工具

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Size](https://img.shields.io/badge/size-~15KB-brightgreen.svg)

## ✨ 功能特性

- **格式化美化** - 一键将压缩的 JSON 格式化为易读的缩进格式
- **压缩优化** - 去除空格和换行，最小化 JSON 文件大小
- **语法验证** - 快速检测 JSON 语法错误，精确定位问题
- **格式转换** - 支持转换为 XML、YAML 等多种格式
- **一键复制** - 快速复制处理结果
- **实时统计** - 显示字符数、行数和文件大小

## 🚀 快速开始

### 本地运行

```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8000

# 或者使用 Node.js
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

### 部署到 GitHub Pages

1. 创建新的 GitHub 仓库
2. 上传 `index.html` 文件
3. 进入仓库 Settings → Pages
4. 选择 main 分支作为源
5. 访问 `https://你的用户名.github.io/仓库名`

### 部署到 Netlify

1. 登录 [Netlify](https://netlify.com)
2. 点击 "Add new site" → "Deploy manually"
3. 拖拽 `index.html` 文件上传
4. 几秒钟后即可访问！

## 📖 使用方法

### 格式化 JSON

1. 将 JSON 数据粘贴到左侧输入框
2. 点击"✨ 格式化"按钮
3. 格式化后的 JSON 会显示在右侧

### 压缩 JSON

1. 将 JSON 数据粘贴到输入框
2. 点击"📦 压缩"按钮
3. 压缩后的最小化 JSON 会显示在输出框

### 验证 JSON

1. 将 JSON 数据粘贴到输入框
2. 点击"✅ 验证"按钮
3. 系统会显示 JSON 是否有效

### 格式转换

1. 将 JSON 数据粘贴到输入框
2. 点击"📄 转XML"或"📝 转YAML"
3. 转换结果会显示在输出框
